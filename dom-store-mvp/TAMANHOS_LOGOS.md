## 🔧 Se Precisar Mudar

Para alterar tamanhos, edite apenas o atributo `height` inline:

```html
<!-- Exemplo: aumentar logo do header -->
<img src="assets/logo-dom.png" alt="DOM Store" style="height: 40px;">
```

O `width` é calculado automaticamente para manter a proporção!

## 🎨 Proporções dos Logos
